# Sentiment-Analysis-PowerBI
# Customer Sentiment Analysis Dashboard (Power BI + Python)

## Project Overview

This project analyzes customer product reviews and performs **sentiment analysis using Python (TextBlob)**.
The results are visualized using **Power BI interactive dashboards**.

## Tools Used

* Power BI
* Python
* TextBlob
* Excel
* DAX

## Dataset

The dataset contains product reviews with the following columns:

* ReviewID
* Date
* Product
* Customer
* Region
* ReviewText

Python is used to classify reviews into:

* Positive
* Negative
* Neutral

## Dashboard Features

The Power BI dashboard includes:

1. KPI Cards

   * Total Reviews
   * Positive Reviews
   * Negative Reviews

2. Sentiment Distribution (Donut Chart)

3. Product Sentiment Analysis (Stacked Column Chart)

4. Region Sentiment Analysis

5. Sentiment Trend Over Time
